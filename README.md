# Data Cleaning and Analysis

## Project Overview
This project focuses on data cleaning, preprocessing, and data analysis using Microsoft Excel. The dataset was analyzed using various Excel formulas and functions to generate meaningful insights and improve data understanding.

## Objectives
- Clean and organize raw data
- Perform data analysis using Excel formulas
- Generate insights from the dataset
- Improve accuracy and readability of data

## Tools Used
- Microsoft Excel

## Excel Functions Used
- SUM
- COUNT
- AVERAGE
- MINIMUM
- MAXIMUM
- IF Function
- SUMIF
- COUNTIF
- LEFT
- RIGHT
- MID

## Project Files
- Raw Data Set
- Worked Data Exploration
- Project Report

## Key Learnings
Through this project, data cleaning techniques, formula usage, and analytical calculations were performed effectively using Excel functions.

## Conclusion
This project helped in understanding how Excel functions can be used for data cleaning, preprocessing, and extracting meaningful insights from raw datasets.
